Web GitHub.io

  1. Buat repository baru di GitHub dengan nama <username>.github.io. <username> menandakan username GitHub. Besar kecil huruf harus sama!
  2. Clone repository.
  3. Bikin file index.html dengan konten HTML untuk tampilan depan website.
  4. Commit & Push.
  5. Tunggu sekitar 30 detik.
  6. Buka <username>.github.io di browser.
